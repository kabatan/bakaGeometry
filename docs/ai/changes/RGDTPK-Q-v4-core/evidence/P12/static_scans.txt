Static scan 2: implementation anchors

Command:

```text
rg -n "squarefree_support|isolate_real_roots|isolate_real_roots_sturm_with_max_width|RootIsolationOptions|decode_candidates|hash_target_candidate|AlgebraicRootRecord|algebraic_roots_from_support|refine_algebraic_root_to_width|compare_algebraic_roots|candidate_hash|finalize_candidate_cover_result|verify_roots_and_candidates|p12_" geosolver-core/src/roots geosolver-core/src/algebra/real_root.rs geosolver-core/src/verify geosolver-core/src/compose/final_support.rs geosolver-core/src/types/rational.rs geosolver-core/tests
```

Required anchors observed:

- `squarefree_support` in `geosolver-core/src/roots/squarefree.rs` rejects zero support and delegates exact squarefree normalization to univariate gcd/derivative logic.
- `RootIsolationOptions` and `isolate_real_roots` in `geosolver-core/src/roots/isolate.rs` connect the public P12 roots API to exact Sturm/Descartes-selected isolation.
- `isolate_real_roots_sturm_with_max_width` in `geosolver-core/src/algebra/real_root.rs` provides exact rational refinement with a positive rational width bound.
- `TargetCandidate`, `decode_candidates`, and `hash_target_candidate` in `geosolver-core/src/roots/decode.rs` bind candidate target, support hash, root index, isolating interval, and candidate hash.
- `finalize_candidate_cover_result` in `geosolver-core/src/compose/final_support.rs` calls `squarefree_support`, `isolate_real_roots`, and `decode_candidates` to construct a production candidate-cover result with exact roots and decoded candidates.
- `AlgebraicRootRecord`, `algebraic_roots_from_support`, `refine_algebraic_root_to_width`, and `compare_algebraic_roots` in `geosolver-core/src/roots/algebraic_number.rs`.
- `hash_decoded_candidates` and `verify_roots_and_candidates` include candidate interval/hash binding in replay.
- `p12_replay_rejects_candidate_omission_and_duplicates_even_when_hashes_match` proves replay rejects omitted or duplicate candidates even when run hashes are recomputed.
- `geosolver-core/tests/p12_roots_decode_integration.rs` proves candidate-cover finalization returns exact root isolation and non-placeholder decoded candidates.
- P12 focused tests cover zero/repeated squarefree, no-real root, one rational root, multiple rational roots, irrational roots, repeated roots, high-coefficient roots, candidate decode binding, algebraic root hash binding, comparison, and refinement.
