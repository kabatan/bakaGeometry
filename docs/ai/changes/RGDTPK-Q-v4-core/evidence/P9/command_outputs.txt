- p9_regular_chain_guard_binding_is_operational verifies guard binding, execution mismatch after guard deletion, and certificate-hash change after guard mutation.

NormTrace:
- algebra/norm_trace.rs now has TowerPlanDescription with ordered TowerStep entries.
- detect_explicit_tower_plan detects multi-step algebraic towers by polynomial form, not geometry names.
- norm_relation_for_tower_plan eliminates each tower variable by sparse resultants.
- verify_norm_tower_plan_relation recomputes the relation exactly.
- p9_norm_trace_multistep_tower_exports_verified_support verifies a two-step tower alpha^2-2, beta^2-alpha, T-beta produces T^4-2 up to sign.
